# Memorize
A lightweight notepad

嫌弃微软便签，自己写了一个便签，非常轻量。可以自己控制便签窗口是否置顶，且带有自动保存和防崩溃保存，可以ctrl+鼠标滚轮缩放字体

Disliking Microsoft Sticky Notes, I wrote my own note app, which allows you to control whether the note window stays on top, and it features auto-save and crash-proof saving. You can also zoom the font with Ctrl + mouse wheel.
